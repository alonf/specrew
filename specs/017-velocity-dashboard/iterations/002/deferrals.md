# Iteration 002 Forward-Looking Deferrals

Feature 017 intentionally stops at the bounded v1 dashboard slice.

## Deferred follow-ups

1. **Full multi-developer aggregation**  
   `--team` remains a friendly fallback until Proposal 010 / multi-developer
   reconciliation lands.

2. **Browser / HTML visualization**  
   The dashboard stays console-first and artifact-first in v1. No browser UI,
   hosted status page, or HTML export is introduced here.

3. **Dense analytics and predictive prioritization**  
   Burndowns, scatterplots, probabilistic forecasts, and "next likely feature"
   hints remain out of scope.

4. **Broader quality-drift automation**  
   Feature 017 adds dashboard-specific roadmap and artifact warnings only. Wider
   governance automation should expand in a separate approved slice.
